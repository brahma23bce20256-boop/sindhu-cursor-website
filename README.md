# Sindhu Restaurant Website

A premium, award-worthy restaurant website for **Sindhu** — featuring smooth scroll animations, parallax effects, and an elegant fine-dining aesthetic.

## Features

- **Smooth Scrolling** — Lenis-powered buttery scroll experience
- **Scroll Animations** — Framer Motion reveal, parallax, and hover effects
- **Premium Design** — Dark charcoal & gold palette with Cormorant Garamond typography
- **Fully Responsive** — Mobile-first design with animated mobile navigation
- **Sections** — Hero, About, Menu, Experience, Gallery, Testimonials, Reservations, Footer

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion
- Lenis (smooth scroll)
- Lucide React (icons)

## Customization

- Replace Unsplash image URLs with your client's actual restaurant photos
- Update menu items, prices, and contact details in the component files
- Modify colors in `tailwind.config.ts` under the `sindhu` palette

## Build for Production

```bash
npm run build
npm start
```
